#Title;
Movie-Recommendation-and-Rating-Visualization-System-

#About and Objective;
The Movie Recommendation System is a Python-based data visualization project that helps users discover movies based on their preferences for genre, country, and language.
The system filters a dataset of movies based on the users input and presents recommendations through interactive visual charts, making movie selection easier and more intuitive.

#Problem Statement;
With the growing availability of movies across different regions and languages, users often struggle to find content that matches their specific preferences.
This project addresses the problem by offering a simple, efficient filtering and visualization system that narrows down choices and highlights top-rated movies.

#Technologies Used;
Programming Language(Python), Libraries(pandas, matplotlib).

#Output;
Matching Movies sorted by ratings with virtual charts
